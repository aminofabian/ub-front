Palmart Till Print Bridge
=========================

Lets the cloud cashier print to a USB / local / network thermal printer
on THIS computer.

1. Unzip this folder.
2. Run the installer for your OS (below).
3. If asked, install Node.js LTS from https://nodejs.org/ then re-run.
4. Return to Palmart Cashier → Printer → Detect printers.

Windows (install once — recommended)
------------------------------------
Double-click: Install-Palmart-Print-Bridge.cmd
(Allow SmartScreen / Run anyway if prompted)

That:
  - installs under %LOCALAPPDATA%\Palmart\till-print-bridge
  - starts the bridge hidden in the background
  - auto-starts at every Windows sign-in

You do NOT need to keep a window open while cashiering.
Manual start (only if needed): start-till-print-bridge.cmd
Uninstall autostart: run Uninstall-Palmart-Print-Bridge.ps1 in the install folder.

Windows 7
---------
Use the separate package: palmart-till-print-bridge-windows7.zip
(No Node.js - PowerShell bridge). See that zip INSTALL.txt.

macOS
-----

Double-click: Install Palmart Print Bridge.command
(If macOS blocks it: right-click → Open)

Linux
-----
bash install-palmart-print-bridge.sh

After install, the bridge listens on http://127.0.0.1:19500 and starts at login.
